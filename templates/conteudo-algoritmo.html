Exercícios de Fixação

Conceito Chave

1. Qual a característica essencial que distingue um algoritmo de uma simples lista de tarefas?

A) O uso de números e fórmulas matemáticas.
B) Ser finito, não ambíguo e produzir um resultado após um número limitado de passos.
C) Ser escrito exclusivamente em linguagem de programação.
D) Conter, obrigatoriamente, um loop de repetição.

Estruturas Lógicas

2. Em termos de estrutura, qual é o elemento que permite que um algoritmo tome decisões com base em uma condição (e.g., 'Se o aluno passou na média, então APROVAR, senão REPROVAR')?

A) Sequência (Instruções em ordem)
B) Decisão/Condicional (Estrutura SE-ENTÃO-SENÃO)
C) Repetição/Iteração (Estrutura ENQUANTO)
D) Subrotina (Função modular)

Visualização

3. Qual das seguintes ferramentas é mais eficaz para representar visualmente o fluxo lógico, as decisões e a ordem de execução dos passos de um algoritmo?

A) Um Mapa Mental.
B) Uma Tabela de Dados.
C) O Código Binário.
D) O Fluxograma.

Eficiência

4. A eficiência de um algoritmo é uma métrica crucial. O que a "eficiência" geralmente mede em termos computacionais?

A) O quão difícil é para um ser humano ler o código.
B) O número total de linhas de código que ele possui.
C) O tempo de execução e/ou a quantidade de memória utilizada para processar os dados.
D) A beleza estética do seu fluxograma.

Interconexão

5. O Algoritmo é o pilar final que usa os resultados dos pilares anteriores. O que o Algoritmo transforma diretamente?

A) A Decomposição em Padrões.
B) O Problema Complexo em Abstrações.
C) O Modelo Abstrato em uma solução executável (passos ordenados).
D) O Código de Programação em Linguagem Natural.

Questão de Projeto: Criando o Algoritmo

Com base no seu modelo abstrato, defina os passos sequenciais, condicionais e/ou de repetição (o algoritmo) que, quando executados, resolverão o seu problema.
